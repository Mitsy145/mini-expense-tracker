#  Mini Expense Tracker

> Exercise 2 — Full Stack Take-Home Assignment | Studio Graphene

## Project Description

Mini Expense Tracker is a full-stack web application built as part of the Studio Graphene Associate Software Engineer take-home assignment (Exercise 2). It allows a single user to log their daily expenses across categories such as Food, Transport, Bills, Entertainment, and Other. The app provides a real-time summary panel showing total spent this month, spending per category, highest single expense, and a live balance calculated against the user's set income. Users can filter expenses by category and date range, visualize spending through a pie chart, and export visible expenses as a CSV file.
